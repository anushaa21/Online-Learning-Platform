# Online-Learning-Platform
A full-stack web application developed using Java technologies that allows students to browse, enroll, and learn from online courses. The platform provides an interactive and user-friendly interface for managing courses, users, and learning content.

Features
User Registration and Login
Course Listing and Details Page
Student Enrollment System
Add to Cart / Purchase Courses
Responsive Frontend Design
Secure Backend APIs
Database Integration for Storing Users and Courses
Admin Management Features
Technologies Used
Frontend: HTML, CSS, JavaScript, React
Backend: Java, Spring Boot
Database: MySQL
Build Tool: Maven
Version Control: Git & GitHub
Project Objectives
Build a complete full-stack learning platform
Understand frontend-backend integration
Implement REST APIs using Spring Boot
Manage course and student data efficiently
Gain practical experience in full-stack Java development
Future Enhancements
Video Course Upload
Payment Gateway Integration
Certificates for Completed Courses
Live Classes and Chat Support
Progress Tracking Dashboard
